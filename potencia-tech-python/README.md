## 💪💻 Desafios  do Bootcamp
> 1. [Criando um sistema bancário com Python](https://github.com/HiroNi9/desafios-dio/blob/main/potencia-tech-python/sistema-bancario-v1.py)
> > ### Descrição do desafio
> > > Criar um sistema bancário com as operações: sacar, depositar e visualizar extrato. Essa versão trabalhara somente com 1 usuário.
> > > 
> > > Regras para saque => Permitido somente 3 saques diários e com limite maximo de R$500,00 por saque, e exibir uma mensagem caso o saldo do usuario seja insuficiente.
> > >
> > > Regras para depósito => Depositar somente valores positivos.
> > >
> > > Regras para extrato => Deve listar todas as operações de saques e depósitos realizadas e se não tiver operações, exibir "Não foram realizadas movimentações". No fim da listagem, exibir o saldo atual da conta. Formatar todos os valores como R$ XXX.XX
